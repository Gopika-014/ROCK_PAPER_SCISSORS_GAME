## Overview

Rock Paper Scissors is a browser-based interactive game built using HTML, CSS, and JavaScript.  
The player competes against the computer in a five-round match where scores are tracked in real time and a final winner is announced.

## Features

- Interactive Rock Paper Scissors gameplay
- Random computer opponent
- Five-round match system
- Real-time score tracking
- Final winner announcement
- Confetti animation when the player wins
- Responsive design for desktop and mobile devices

---

## How to Play

1. Click on Rock, Paper, or Scissors
2. Computer randomly selects its move
3. Each round updates the score
4. After 5 rounds, final winner is displayed
